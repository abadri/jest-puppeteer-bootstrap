# jest-puppeteer-bootstrap

## Prerequsites
1. NodeJS >= v12.0.0

## Setup

```bash
git clone git clone https://github.com/abadri/jest-puppeteer-bootstrap.git
cd jest-puppeteer-bootstrap
npm install
```

## Test file location
All the end to end tests are located under `tests` folder.

## How to run tests

```bash
npm test
```
